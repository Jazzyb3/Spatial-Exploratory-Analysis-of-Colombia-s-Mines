# English

**legal_mines_all** has all the data, even if there are missing or NA for coordinate points

**legal_mines_1** has all of the digital file mines that contain coordinate points

**legal_mines_2** has all paper file mines that contained coordinate points. These also might be off due to the interpretation of the handwriting, and changes to the data had been made to attempt to match the coordinate format/pattern that was observed.

# Spanish

**legal_mines_all** contiene todos los datos, incluso si faltan puntos de coordenadas o si no se especifican.

**legal_mines_1** contiene todas las minas de archivos digitales que contenían puntos de coordenadas.

**legal_mines_2** contiene todas las minas de archivos en papel que contenían puntos de coordenadas. Estos datos también podrían estar erróneos debido a la interpretación de la escritura a mano y a cambios en los datos realizados para intentar que coincidan con el formato/patrón de coordenadas observado.
